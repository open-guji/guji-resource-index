# 香港科技大学图书馆 - Rare & Special e-Zone

香港科技大学图书馆的"珍稀特藏电子区"（Rare & Special e-Zone），提供馆藏线装古籍、民国书刊、古地图等特藏资源的数字化在线阅览。

- 网站链接：https://lbezone.hkust.edu.hk/rse/
- 分享协议：待确认（网站未明确标注）
- 约 2,000+ 册线装古籍（清代及民国时期）
- 资源类型：线装古籍、民国书刊、古地图、摄影集
- 不支持 IIIF 协议
- bookget wiki 中有收录，但无专用下载模块

## 搜索与浏览

- 搜索页面：https://lbezone.hkust.edu.hk/rse/?s=*&scopename=chinese-collection
- 在页面顶部搜索框输入关键词即可搜索
- 可按专题分类浏览：
  - [线装古籍（Thread Bound）](https://lbezone.hkust.edu.hk/rse/thread-bound-books)
  - [民国书刊（Chinese books before 1949）](https://lbezone.hkust.edu.hk/rse/chinese-collection)
  - [古地图（Antique Maps of China）](https://lbezone.hkust.edu.hk/rse/?page_id=6)
  - [东亚古地图（Ancient East Asian Maps）](https://lbezone.hkust.edu.hk/rse/ancient-east-asian-maps)
- 无需注册登录

# 开发者文档

## URL 结构

### 详情页
```
https://lbezone.hkust.edu.hk/bib/{bib_id}
```
- `bib_id`：书目编号，如 `b1129168`
- 实际会重定向到 WordPress 文章页：`https://lbezone.hkust.edu.hk/rse/?p={post_id}`

### BookReader 阅读器
```
https://lbezone.hkust.edu.hk/bookreader/view.php?obj={object}&pgrs={page_progression}&up=2
```
- `obj`：对象标识符
- `pgrs`：翻页方向
- `up`：显示模式（2 = 双页）

### 搜索
```
https://lbezone.hkust.edu.hk/rse/?s={query}&scopename={collection}
```
- `scopename`：集合名称，如 `chinese-collection`、`thread-bound-books`

## 搜索 API

网站基于 WordPress + 自定义插件 `dlib-search`，无公开 JSON API。搜索通过 WordPress 原生搜索实现。

支持自动补全：
```
GET https://lbezone.hkust.edu.hk/rse/?method=autocomplete&term={query}
```

## 元数据获取

网站为传统服务端渲染（WordPress），元数据直接在 HTML 中。

### og:description 元数据

最方便的提取方式是解析 `<meta property="og:description">` 标签，其中以空格分隔包含所有字段值：
```
Chinese books before 1949 (Thread Bound) 中國實用藥物學 趙賢齊編述 book 中國醫藥硏究會 民國20 [1931] ...
```

### HTML 结构

元数据以内联样式的 `<h3>` 和 `<div>` 标签呈现：
- 题名：`h3.post-title`
- 著者：`h3.post-title` 后的下一个 `<h3>`（`color:#737373`）
- 年代：`span` 内（`color:#737373; font-weight:700`）
- 形态：`div`（`line-height:19px; color:#737373; font-weight:700`）

### 字段映射

| 中文字段 | 提取方式 |
|---------|---------|
| 题名 | `h3.post-title` |
| 著者 | `.post-title` 后续兄弟 `h3` |
| 年代 | `span[style*="font-weight:700"]` |
| 形态 | 年代后续 `div[style*="font-weight:700"]` |
| DOI | og:description 中 `10.14711/spcol/{bib_id}` |
| Bib ID | URL 中 `/bib/{bib_id}` |

## 下载

### bookget 支持

bookget wiki 中收录了此网站，但 `app/` 目录下无专用 `.go` 文件（无 `hkust.go` 或 `lbezone.go`），可能使用通用下载模块。

### 图片获取

图片通过 BookReader 组件加载，具体图片 URL 结构待确认。BookReader iframe 内嵌于详情页中。

- 无 IIIF manifest
- 无已知的图片保护措施（待确认）
